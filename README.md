# Dailylaid

> 🤖 基于 QQ 机器人的个人日常事务 AI Agent

## 功能

- 📝 **人情追踪** - 记录谁帮了你、你帮了谁
- ✅ **待办管理** - 自然语言添加任务
- 🌟 **兴趣收集** - 记录感兴趣的博主/领域/事物
- 📥 **收集箱** - 暂存未识别的消息

## 技术栈

- **QQ 通信**: NapCat
- **后端框架**: Flask (Python)
- **AI 大脑**: OpenAI 兼容 API
- **数据存储**: SQLite

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env

# 启动服务
python app.py
```
